# Core Principle


Customer Value is different from Buying Opportunity.


The system must separate:


1. Customer Value Score

Question:

"Is this customer worth building a long-term relationship with?"


2. Buying Opportunity Score

Question:

"Is this customer likely to generate business in the near future?"


3. Sales Priority Score

Question:

"Who should sales contact first today?"



# 1. Customer Value Model


Design the long-term customer evaluation model.


Factors:

- Customer type
- Industry fit
- Product fit
- Company size
- Business model
- Import capability
- Historical relationship
- Long-term cooperation potential


Example:


Hospitality Furniture Distributor:

High Customer Value


Small restaurant startup:

Lower Customer Value but may have high short-term opportunity.



# 2. Buying Opportunity Model


Design short-term opportunity evaluation.


Important:

Buying timing is NOT customer value.

It only affects:

- sales urgency
- follow-up priority


Signals:


- New project
- Expansion
- Renovation
- Supplier search
- RFQ
- Website signals
- Social media signals
- Direct inquiry
- Contact response



If no timing data exists:

The system must not guess.


Example:


Purchase Timing:

Unknown


Confidence:

Low



# 3. Sales Priority Model


Design a combined action priority model.


Example:


Sales Priority =
Customer Value
+
Buying Opportunity


The system should explain:

Why this customer is prioritized.



# 4. Industry Dynamic Scoring


The scoring model must not be fixed globally.


Different industries require different models.


Example:


Hospitality Furniture:


Important:

- Distributor type
- Design company
- Import ability
- Project signals


Cosmetic Packaging:


Important:

- Brand growth
- New product launch
- Packaging replacement


Hotel Furniture:


Important:

- Project stage
- Design company involvement
- Hotel expansion



Design:

Industry Scoring Template.


Each industry can have:

- scoring factors
- weights
- rules



# 5. AI Learning Adjustment


Design how scoring improves over time.


The system should learn from:

Sales feedback:

- Interested
- Not interested
- Wrong customer
- Purchased
- Lost opportunity


Business results:

- Quote created
- PI created
- Order completed


AI can recommend:

- weight adjustment
- scoring improvement


But:

Do not automatically change critical scoring rules without approval.
