# Modules

## Module 01 — Platform Foundation

Application shell, navigation, authentication, roles, dashboard, and local deployment foundation.

## Module 02 — Core Foundation Data

Shared configuration, tags, media types, prompts, administration, sorting, activation, and role restrictions.

## Module 03 — Product Library

Product list/detail/create/edit, automatic and manual unique SKU handling, categorized multi-tags, search, filtering, and media support.

## Module 04 — Product Knowledge Engine V2

Suitable store types and styles, features, customer types, related products and cases, AI-ready fields, Knowledge Score, Knowledge Dashboard, and combined search.
