# Roadmap

## Accepted

- Module 01: Platform foundation
- Module 02: Core foundation data
- Module 03: Product library, SKU rules, tags, search, and media
- Module 04: Product Knowledge Engine V2

## Current Focus

- Production database connectivity
- Migration verification
- Stable cloud deployment and acceptance

## Future

Module 05 has not started. Its scope requires separate product-manager approval.

Potential later work includes proposal automation, CRM integration, AI APIs, imports, and infrastructure scaling. These items are not authorized by this roadmap alone.
