## Convert to Customer

Conversion copies:

- Company information
- Contact information
- Customer type
- Opportunity information

The original Search Result is preserved and marked:

- `converted`

The new Customer record keeps:

- Customer type
- Industry
- Opportunity notes
- AI summary
- AI recommendation
- Opportunity score

## Duplicate Protection

Before converting, the system checks:

- Same website
- Or same company name + country

If a possible duplicate is found, the system returns:

`Possible existing customer found.`

The system does not automatically merge records.
